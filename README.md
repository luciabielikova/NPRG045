# ZooApp

## Requirements
- PHP 8.1 or higher
- Composer

## Installation

1. Clone repository or download .ZIP:
```bash
git clone https://github.com/luciabielikova/zooApp.git
cd zooApp
```

2. Install dependencies
```bash
composer install
```

3. Run
```bash
php -S localhost:8000
```

4. Open in browser http://localhost:8000
